NAME

http_setTimeout - Changes the maximum amount of time HTTP Fetcher will wait on data.
 

SYNOPSIS

#include <http_fetcher.h>

void http_setTimeout(int seconds);
 

DESCRIPTION

HTTP Fetcher will wait seconds until timing out. The default value is 30 seconds. If the timeout value elapses without more data from the server, http_fetch will return with an error.

If you pass a value less than 0, reads will not time out.
 

RETURN VALUE

http_setTimeout returns no value and should always succeed.
